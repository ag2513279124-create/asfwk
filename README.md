주식 분석 플러그인 개인정보 보호정책 (간략판)
본 주식 분석 플러그인(이하 "플러그인")은 한국 개인정보보호법 및 관련 금융 규정을 준수하며, 사용자의 개인정보를 보호하기 위해 다음과 같은 정책을 시행합니다. 본 정책은 플러그인 사용 시 적용됩니다.

1. 수집하는 개인정보의 종류 (Types of Collected Personal Information)
플러그인은 다음 정보만 수집하며, 추가적인 개인정보는 수집하지 않습니다：

사용자가 직접 입력한 주식 코드 (단, 본 정보는 로컬 브라우저에만 저장되며 서버로 전송되지 않음)
플러그인 사용 기록 (브라우저 로컬에서만 저장, 금융 정보와 연계하지 않음)
2. 개인정보의 사용 목적 (Purpose of Use)
수집된 정보는 오직 주식 정보 조회 및 분석 기능 제공에만 사용되며, 다음 목적 외에는 사용되지 않습니다：

사용자 입력 주식 코드 기반 주식 정보 제공
플러그인 기능 개선을 위한 통계적 분석 (개인 식별이 불가능한 형태로만 분석)
3. 개인정보의 보유 및 파기 (Retention and Destruction)
사용자 입력 정보는 브라우저 로컬에 최대 30일 보유 후 자동 삭제되며, 사용자가 플러그인을 삭제할 경우 즉시 모든 로컬 정보가 파기됩니다. 서버에 저장된 통계 데이터는 개인 식별이 불가능한 형태로만 보유합니다.

4. 개인정보의 제3자 제공 (Disclosure to Third Parties)
본 플러그인은 사용자의 개인정보를 어떠한 제3자에게도 제공하지 않으며, 금융 기관·제3기업과의 정보 공유도 실시하지 않습니다.

5. 사용자의 권리 (User's Rights)
사용자는 언제든지 본인의 개인정보 조회·수정·삭제를 요청할 수 있으며, 플러그인 설정에서 직접 로컬 저장 정보를 삭제할 수 있습니다. 권리 행사 문의는 아래 연락처로 하시기 바랍니다.

6. 보호정책의 변경 (Policy Changes)
본 보호정책은 법령 변경 또는 플러그인 기능 개선에 따라 수정될 수 있으며, 변경 시 플러그인 내 공지 후 적용됩니다.

7. 문의처 (Contact)
개인정보 보호 관련 문의는 아래 이메일로 보내주시기 바랍니다：
